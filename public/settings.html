To configure environment variables:

  1. Go to Netlify Environment Variables
  2. Click "Add a variable"
  3. Add the required variables (see docs)
  4. Click "Trigger deploy" to apply changes

Required Variables:

  • META_ACCESS_TOKEN - Meta Ads API token
  • META_AD_ACCOUNT_ID - Meta ad account ID (act_XXX)
  • GOOGLE_CLIENT_ID - OAuth Client ID
  • GOOGLE_CLIENT_SECRET - OAuth Client Secret
  • GOOGLE_REDIRECT_URI - OAuth redirect URL
  • GOOGLE_SPREADSHEET_ID - (Optional) Sheets ID
